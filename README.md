# Caffe-Web 
### ระบบเว็บไซต์ร้านกาแฟ (Demo Frontend Project)  
รายวิชาที่เกี่ยวข้อง: Web Programming / Software Engineering / UI Development  

---

##  1. บทนำ (Introduction)

Caffe-Web เป็นโปรเจกต์ตัวอย่างที่สร้างขึ้นเพื่อฝึกการพัฒนาเว็บไซต์ด้วย  
**HTML, CSS และ JavaScript** ตามที่อาจารย์มอบหมาย  
โดยมีเป้าหมายให้ผู้เรียนเข้าใจพื้นฐานของการสร้าง Web UI, การจัดวาง Layout,  
รวมถึงการออกแบบประสบการณ์ผู้ใช้ (UX/UI) สำหรับร้านกาแฟแบบออนไลน์

โปรเจกต์นี้ครอบคลุมการสร้างหน้าเว็บหลักของร้านกาแฟ เช่น  
หน้า Home, เมนูเครื่องดื่ม, โปรโมชั่น และหน้า Contact

---

## 2. วัตถุประสงค์ของโปรเจกต์ (Objectives)

1. เรียนรู้พื้นฐานการสร้างเว็บไซต์ด้วย HTML/CSS/JS  
2. ทดลองออกแบบ UI ให้สวยงามและเหมาะสมกับร้านกาแฟ  
3. ทำความเข้าใจการจัดองค์ประกอบหน้าเว็บ (Layout Structure)  
4. ฝึกการจัดการไฟล์ Frontend แบบเป็นระบบ  
5. ฝึกการอัปโหลดโปรเจกต์ขึ้น GitHub  

---

##  3. โครงสร้างโปรเจกต์ (Project Structure)

```

Caffe-Web/
│
├── css/ # ไฟล์ตกแต่งหน้าเว็บ (CSS)
│ └── ...
│
├── js/ # ไฟล์ JavaScript สำหรับ interaction
│ └── ...
│
├── img/ # รูปภาพที่ใช้ในเว็บไซต์
│ └── ...
│
├── index.html # หน้า Landing Page / หน้าแรกของระบบ

├── login-user.html # หน้าเข้าสู่ระบบ
├── register-user.html # หน้าสมัครสมาชิก
├── menu-user.html # หน้ารายการเมนูเครื่องดื่มและอาหาร

├── promotion-user.html # หน้าโปรโมชั่นต่าง ๆ
├── review-user.html # หน้ารีวิวจากลูกค้า
├── queue-user.html # หน้าแสดงลำดับคิวลูกค้า
├── order-admin.html # หน้าสำหรับผู้ดูแลระบบจัดการคำสั่งซื้อ
├── navAndfooter-user.html # ส่วน Navbar และ Footer ที่นำไป include
│
└── README.md # เอกสารอธิบายโปรเจกต์

```

##  4. หน้าที่ของเว็บไซต์ (Website Features)

###  หน้า Home
- นำเสนอหน้าตาร้านและแบรนด์  
- มีข้อความต้อนรับและภาพตกแต่งให้ดึงดูดสายตา  

###  หน้าเมนู (Menu Page)
- แสดงรายการกาแฟ เช่น Espresso, Latte, Cappuccino  
- ใช้ Grid Layout และ Flexbox เพื่อจัดตำแหน่งเนื้อหา  

###  หน้าโปรโมชั่น / คำแนะนำ (Optional ตามไฟล์)
- จัดแสดงสินค้าแนะนำหรือเมนูยอดนิยม  

###  หน้า Contact / Feedback
- มีข้อมูลช่องทางติดต่อร้าน  
- แบบฟอร์มส่งข้อความ (ถ้ามี)

##  6. วิธีรันโปรเจกต์ (How to Run)

1. Clone โปรเจกต์จาก GitHub  
```

git clone [https://github.com/teetayu/Caffe-Web](https://github.com/teetayu/Caffe-Web)

```

2. เปิดโฟลเดอร์ใน VS Code  

3. เปิดไฟล์  
```

index.html

```
ด้วย Browser เช่น Chrome หรือ Edge

ไม่ต้องติดตั้ง library เพิ่มเพราะเป็นโปรเจกต์แบบ Static Website

##  8. สิ่งที่ได้เรียนรู้จากงานนี้ (What I Learned)

###  1) พื้นฐานการสร้างหน้าเว็บด้วย HTML/CSS  
- การจัดหน้าเว็บให้เป็นส่วนหัว ส่วนเนื้อหา และส่วนท้าย  
- การใช้สี ฟอนต์ และภาพให้เหมาะสมกับธีมร้านกาแฟ

###  2) การออกแบบ UX/UI  
- เรียนรู้การจัดองค์ประกอบให้ผู้ใช้เข้าใจง่าย  
- ทำปุ่มและเมนูนำทาง (Navigation) ให้ใช้งานสะดวก

###  3) JavaScript เบื้องต้น  
- การเพิ่มความโต้ตอบ เช่น ปุ่มคลิก, การเลื่อนหน้าเว็บ  
- ฝึกเขียน script แบบง่าย ๆ ให้ทำงานร่วมกับ HTML/CSS

###  4) การใช้งาน Git & GitHub  
- วิธี clone, commit, push, pull  
- การจัดโครงสร้างไฟล์เพื่อให้โปรเจกต์ดูเป็นระเบียบ  
- เรียนรู้ความสำคัญของ README ในงานจริง  

โดยรวมโปรเจกต์นี้ช่วยสร้างพื้นฐานสำคัญสำหรับการทำ Web Application  
และสามารถต่อยอดไปสู่ Frontend Framework เช่น React, Vue หรือ Tailwind ได้ในอนาคต
